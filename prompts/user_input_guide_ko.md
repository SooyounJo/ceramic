## 사용자 입력 가이드 (CMF 재질 생성)

아래 4가지만 채우면 됩니다. (나머지는 기본값이 자동 적용)

1) **재질(Material)**: 예) 유약 세라믹, 브러시드 알루미늄, 소프트터치 폴리머
2) **색(Color)**: 예) 차콜 블랙 + 쿨 블루 언더톤, 저채도 실버, 웜 라이트 그레이
3) **마감(Finish)**: 예) 매트, 새틴(반광), 하이글로스, 아노다이징, 브러시드
4) **표면 디테일(텍스처/요철)**: 예) 미세 오렌지필, 미세 스크래치, 미세 그레인, 샌드블라스트

선택 입력:
- **조명(Lighting)**: 예) 좌측 상단 키라이트 + 소프트 3점 조명, 그림자 최소화
- **추가 요청(Optional)**: 예) 더 선명한 에지 하이라이트, 반사 강도 낮게 등

기본 고정 조건(자동 적용):
- 공중에 **구(스피어) 1개만** 중앙에 떠 있음
- **순백 배경**, **바닥/접촉 그림자 없음**
- **텍스트/로고/라벨/다른 오브젝트 없음**
